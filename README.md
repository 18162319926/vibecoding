
# 织伴

织伴是一个专为编织爱好者设计的轻量级 Web 应用，聚焦于项目进度管理、毛线库存、小样记录、数据统计、图片导出与多端同步。支持离线本地存储与云端实时同步，兼容桌面与移动端。

---

## 主要功能

### 1. 项目仪表盘（首页）
- 项目状态筛选：全部 / 进行中 / 搁置 / 已完成
- 新建项目：空白、导入图片图解、导入文字图解
- 项目卡片：名称、封面、状态、进度、导出风格
- 快捷操作：进入、切换状态、导出图片、删除
- 全局计时器：开始/暂停/重置，状态持久化

### 2. 项目详情页
- 编辑项目信息：名称、类型、状态、总行数、线材、工具、针号等
- 进度操作：行数增减、步进调整、进度百分比
- 图片与图解：封面上传/删除，支持文字与多图图解
- 导出能力：Canvas 导出，支持预览与下载

### 3. 毛线仓库
- 字段：品牌、类型、色号、针号、重量、季节、消耗进度、实拍图
- 操作：新增、编辑、删除
- 视觉：毛线风格进度条（细条纹、低饱和）

### 4. 小样仓库
- 字段：线材、花型、针号、规格、密度、备注、实拍图
- 操作：新增、编辑、删除

### 5. 数据统计页
- 支持周期切换（本周、本月、历史）
- 统计内容：总时长、总行数、活跃天数、平均/最大每日时长与行数、偏好时段分布、趋势图、项目排行等
- 偏好时段分布与统计严格按周期边界，数据口径统一
- 支持数据修复脚本（如自动补全缺失的分时段数据）

### 6. 云同步与账号
- 支持 Supabase 登录注册、会话管理
- 云端实时同步项目、计时器、毛线、小样数据
- 支持多端实时/轮询同步，断网自动切换离线模式
- 账号信息展示，支持随时切换/退出

---

## 存储与同步机制

- 本地：所有数据优先写入 localStorage，离线可用
- 云端：登录后自动同步本地与云端，支持实时监听和手动强制同步
- 图片存储：项目封面图等上传至 Supabase Storage
- 数据表建议：knit_user_state（含 user_id、projects、timer、client_updated_at），开启 RLS 仅允许用户访问自己的数据

---

## 技术栈

- 前端：HTML5 + CSS3 + 原生 JavaScript
- 数据：localStorage + Supabase（Auth / Postgres / Storage / Realtime）
- 导出：Canvas API
- 图片读取：FileReader
- OCR：Tesseract.js（可选，主流程不依赖）

## 使用建议

- 建议注册账号并登录体验云同步
- 未登录时数据仅保存在本地，清理浏览器缓存会丢失
- 登录后自动同步，支持多端实时/轮询
- 图片导出建议在桌面端操作，上传图片建议≤2MB
- 推荐用静态服务器（如 VS Code Live Server）本地运行

---

## 已知问题 / 局限

- 多端同时编辑同一项目可能有同步冲突
- 移动端适配已覆盖主流手机，极少数设备显示可能不完美
- 本地数据未登录时无云端备份，丢失不可恢复
