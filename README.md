
# 织伴

织伴是一款面向编织爱好者的网页端项目管理与素材管理工具，支持项目进度追踪、毛线库存管理、小样（Swatch）管理、全局计时、数据统计等功能。界面简洁，操作直观，适合个人或小型工作室日常使用。

---


## 主要功能
### 0. 报告管理（周报/月报/自定义）
- 自动生成上周/上月统计报告，支持自定义新建、编辑、删除
- 支持按周报、月报筛选浏览，历史报告自动保留
- 支持内容编辑、周期统计、导出等


### 1. 项目管理
- 新建、编辑、删除编织项目
- 记录项目名称、类型、状态、总行数、封面、图解、材料等信息
- 支持项目进度（行数）统计与切换状态
- 项目详情页可累计用时、切换状态、删除项目

### 2. 毛线库存管理
- 新增、编辑、删除毛线条目
- 支持品牌、线材类型、色号、适用针号、原始重量、库存重量、适用季节、消耗进度等字段
- 支持上传毛线实拍图片
- 毛线库存列表支持筛选、编辑、删除

### 3. 小样（Swatch）管理
- 新增、编辑、删除小样条目
- 支持关联线材、花型、针号、规格、针数、行数、备注、实拍图片等字段
- 小样列表支持筛选、编辑、删除

### 4. 全局计时器
- 项目页右侧提供全局计时器，支持自定义分钟数、开始/暂停/重置
- 计时器与项目累计用时联动

### 5. 数据统计
- 支持项目、毛线、小样等多维度数据统计与展示
- 统计页可查看进度、用时等汇总信息

### 6. 账户与同步
- 当前为纯本地模式（无账号系统、无云同步）
- 所有数据保存在浏览器本地存储（localStorage）

## 项目结构

```
织伴项目目录：
├── index.html              # 项目仪表盘（首页）
├── project.html            # 项目详情页
├── yarn.html               # 毛线库存管理页
├── swatch.html             # 小样管理页
├── stats.html              # 数据统计页
├── script.js               # 首页及通用逻辑
├── project.js              # 项目详情页逻辑
├── storage.js              # 毛线/小样仓库页逻辑
├── storage-account.js      # 仓库页账号菜单逻辑
├── report.html             # 报告管理页（周报/月报/自定义报告）
├── report.js               # 报告页逻辑（自动生成、筛选、编辑、删除等）
├── stats.js                # 统计页逻辑与数据修复脚本
├── styles.css              # 全局样式表
├── tesseract.min.js        # OCR 依赖（可选，图片识别）
├── favicon.ico             # 网站图标
├── README.md               # 项目说明文档
└── .vscode/                # VSCode 配置（如端口、任务、思维导图等）
	├── settings.json       # VSCode 本地设置
	├── todos.json          # 任务清单（开发/维护用）
	├── project.km          # 项目思维导图（可选）
	└── site.km             # 站点结构图（可选）
```

## 快速开始
1. 克隆本仓库到本地
2. 直接打开 `index.html` 开始使用
3. 移动端可将网页添加到主屏幕，获得更接近 App 的体验

数据说明：
- 项目数据保存在 `knit-helper-state`
- 计时器数据保存在 `knit-global-timer`
- 毛线仓库数据保存在 `knit-yarn-storage`
- 小样仓库数据保存在 `knit-swatch-storage`
